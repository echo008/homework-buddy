Pain Points

传统听写，谁苦的谁知道

每个有学龄孩子的家庭，都在经历一场 nightly 的「报听写马拉松」。问题不在孩子，而在工具太落后。

😫

家长精力被绑架

每天必须空出 20-30 分钟全程陪同，无法做饭、工作或休息，重复劳动消耗巨大。

🗣️

发音不标准误导

家长英语发音不准、语文声调出错，孩子反而记住错误发音，纠正成本更高。

⏱️

节奏全凭感觉

报太快孩子写不完，报太慢拖时间，没有固定间隔，训练效果打折扣。

📝

批改整理繁琐

人工对答案、手抄错题本，易漏易错，错题无法追踪和针对性重测。

Core Highlights

六大核心能力，逐个击破听写痛点

不只是「把家长替掉」，而是在每一个环节都比传统方式更专业、更稳定、更可追溯。

📷

拍照识字 · 秒建词库

火山引擎 OCR 识别课本照片，支持印刷体、手写标注、倾斜角度，3 秒完成图片到文字的转换。

火山 OCR多场景识别
🧠

AI 清洗 · 结构化入库

豆包大模型自动过滤页码、噪音符号，输出「单词-拼音-释义」标准 JSON，直接写入云数据库。

豆包大模型Prompt 工程
🎲

智能抽题 · 防作弊

支持按单元/课次筛选，设置 Min-Max 数量区间，Fisher-Yates 洗牌打乱顺序，每次题目都不重复。

Min-Max 随机顺序打乱
🔊

标准发音 · 节奏可控

微信同声传译插件 / 大厂 TTS 播报，发音标准、语速稳定，可设置词间等待时间（3s / 5s / 8s）。

标准 TTS间隔控制
✅

自动批改 · 错题本

提交后秒级对比标准答案，错误单词高亮展示，自动归档到错题本，支持一键重测薄弱项。

秒级批改薄弱追踪

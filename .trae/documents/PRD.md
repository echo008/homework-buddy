# 智听 · Homework Buddy Web 版 产品需求文档 (PRD)

## 1. 产品概述

智听是一款面向中小学生及家长的智能听写练习工具，支持英语单词/语文汉字的多模式听写、自定义词库、班级共享、OCR拍照导入等功能。Web 版彻底摆脱微信小程序依赖，实现跨平台（PC/手机/平板）浏览器直接访问，部署简单、开发体验更佳。

- **目标用户**：小学生、初中生、家长、老师
- **核心价值**：随时随地打开浏览器即可听写，无需安装微信/小程序；支持自定义录音、AI 语音播报、拍照导入词库
- **升级亮点**：PWA 可安装、键盘快捷键操作、桌面端大屏体验、数据本地持久化+云端同步

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名/密码或本地游客模式 | 创建词库、听写练习、查看历史记录 |
| 班级创建者 | 注册用户 | 创建班级、生成6位邀请码、管理共享单元、解散班级 |
| 班级成员 | 注册用户 | 加入班级、使用共享词库听写、退出班级 |

### 2.2 功能模块
1. **首页**：听写配置入口（学科、模式、词库选择、数量范围）、快速开始、最近练习记录
2. **听写进行页**：语音播报/显示题目、键盘输入作答、进度显示、暂停/重听/上一题/下一题控制
3. **结果页**：得分统计、错题列表、正确答案对照、错题重测、保存记录
4. **单元管理**：创建/编辑/删除单元、单元列表（我的+班级共享）、单元排序
5. **单词管理**：单词列表、添加/编辑/删除单词、批量导入、自定义录音
6. **班级中心**：创建班级、加入班级（6位码）、班级详情、共享单元管理、成员列表、退出/解散班级
7. **拍照导入**：上传图片、OCR识别、解析预览、确认导入
8. **预置词库**：按学段/教材/版本筛选教材单元、预览单词、直接听写或导入我的词库
9. **个人中心/历史**：听写历史记录、成绩趋势、错题本、个人信息

## 3. 核心流程

```mermaid
flowchart TD
    "A[打开网页/登录]" --> "B[首页配置听写]"
    "B" --> "C{选择词库来源}"
    "C" --> "我的单元" --> "D[选择题数/模式]"
    "C" --> "班级共享" --> "D"
    "C" --> "预置教材" --> "D"
    "C" --> "拍照导入" --> "E[OCR识别单词]" --> "F[导入到我的单元]" --> "D"
    "D" --> "G[开始听写]"
    "G" --> "H[语音播报题目]"
    "H" --> "I[用户输入答案]"
    "I" --> "J{提交判断}"
    "J" --> "正确" --> "K[下一题]"
    "J" --> "错误" --> "L[标记错题]" --> "K"
    "K" --> "M{还有题目?}"
    "M" --> "是" --> "H"
    "M" --> "否" --> "N[结果页]"
    "N" --> "O[查看成绩/错题]"
    "O" --> "P{操作选择}"
    "P" --> "错题重测" --> "G"
    "P" --> "再练一次" --> "G"
    "P" --> "保存记录" --> "Q[历史记录]"
    "P" --> "返回首页" --> "B"
```

听写核心流程：配置 → 获取题目列表 → 逐题播报 → 用户作答 → 即时判定 → 完成统计 → 保存/重测。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：靛蓝 `#4f46e5`（继承品牌色），搭配暖橙 `#f97316` 作为强调色（鼓励/正确反馈）
- **辅助色**：成功绿 `#10b981`、错误红 `#ef4444`、中性灰 `zinc` 系列
- **按钮风格**：大圆角（16px）、大触控区域（最小高度44px）、柔和阴影、hover/active微动效
- **字体**：
  - 标题：'Noto Sans SC', system-ui, sans-serif（中文友好）
  - 英文/数字：'Nunito', 'Noto Sans SC', sans-serif（圆润可爱，适合儿童）
  - 代码/拼音：monospace 字体保证拼音对齐
- **布局风格**：卡片式布局、顶部导航栏（PC端）/底部Tab（移动端自适应）、大量留白、渐变背景装饰
- **图标**：Lucide React 线性图标，风格统一
- **动效**：页面切换淡入、按钮按压缩放、正确/错误即时颜色反馈、进度条平滑过渡

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区 | 渐变背景（靛蓝→紫）、大标题"今天想听写什么？"、浮动装饰元素 |
| 首页 | 模式选择 | 大卡片网格（4种模式卡片），选中态边框高亮+缩放 |
| 听写页 | 题目区 | 超大字号（桌面端48px+）、居中显示、题目类型标签 |
| 听写页 | 作答区 | 大输入框（高64px）、聚焦发光边框、大提交按钮 |
| 结果页 | 成绩概览 | 圆形进度条（正确率）、数字动画从0到目标值 |
| 结果页 | 错题列表 | 错误项红框高亮、用户答案划删除线、正确答案绿色显示 |
| 词库管理 | 单元卡片 | 左色条标识学科、右侧操作按钮三点菜单、hover阴影提升 |
| 班级页 | 班级码 | 等宽字体大号显示、复制按钮、二维码可选 |

### 4.3 响应式设计
- **桌面端优先**：1200px+ 三栏布局（导航/内容/侧边），最大宽度1200px居中
- **平板端**：768px-1199px 两栏布局，卡片间距缩小
- **移动端**：<768px 单栏布局，底部Tab导航，按钮全宽，输入框放大适配触屏
- **触控优化**：所有可点击元素最小44×44px，按钮间距≥8px，支持滑动手势切换题目

### 4.4 PWA 支持
- 可添加到主屏幕/桌面，类App体验
- 离线缓存静态资源，弱网环境可用
- 听写数据本地暂存，联网后同步
